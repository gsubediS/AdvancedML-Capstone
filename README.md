# AdvancedML-Capstone

With increasing technological use and decreasing attention spans, I am interested in exploring a model that is able to predict a user’s distraction patterns using behavioural patterns such as mouse, keyboard and application switching. My goal is to be able to learn individualized patterns that correlate with distraction and create a predictive model that catches and alerts when those patterns are detected.

RQ : Can user interaction patterns be used to accurately predict distraction in real time and for everyday general use?
